# deshmuki-express
This is a simple food delivery application built with Express.js. It allows users to browse available food items, place orders, and manage their profiles.
## Features
- User registration and authentication
- Browse food items by category
- Place and track orders
- User profile management
- Admin panel for managing food items and orders
